# Vera Dashboard

Next.js admin dashboard for the Vera price comparison app.

## Setup

1. Install dependencies:
   ```bash
   bun install
   ```

2. Start the development server:
   ```bash
   bun run dev
   ```

The app will run on `http://localhost:3001`.

## Features

- Submit product URLs for processing
- View trending/recently added products
- Monitor submission status
- View API status

## Backend Integration

The dashboard connects to the Vera backend API on port 3000. Make sure the backend is running before starting the dashboard.
